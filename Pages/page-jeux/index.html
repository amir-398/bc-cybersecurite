DDOS & Adresse IP

Le joueur doit protéger son site web contre une attaque
DDoS. Le joueur doit cliquer sur les attaquants qui
essaient de surcharger le site web avec des demandes
de connexion. Les attaquants veulent faire tomber le site web et l’empêcher de fonctionner convenablement.
Le joueur doit également répondre à des questions sur
les attaques DDoS pour gagner des points
supplémentaires....

Défacement : Différence

Le joueur doit différencier un site officiel qui n'a pas été
hacké d'un site officiel qui a été hacké et dont
l'apparence est différente de l'originale. Le joueur doit
regarder deux images côte à côte, l'une étant
l'apparence originale du site web et l'autre étant
l'apparence modifiée après le défacement. Le joueur
doit cliquer sur l'image qui correspond.
