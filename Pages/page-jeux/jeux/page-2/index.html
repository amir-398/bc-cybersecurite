Devenir maître du phishing

Niveau 1

Le joueur doit protéger son site web contre une attaque DDoS. Le joueur doit cliquer sur les attaquants qui essaient de surcharger le site web avec des demandes de connexion. Les attaquants veulent faire tomber le site web et l’empêcher de fonctionner convenablement.


Niveau 2

Le joueur doit également répondre à des questions sur les attaques DDoS pour gagner des points supplémentaires. Le but du jeu est d'apprendre comment fonctionne une attaque DDoS et comment se protéger contre elle.




Niveau 3

Une fois que les attaquants ont été repoussés par le joueur. Il doit deviner l'emplacement géographique d'une adresse IP donnée. Le joueur doit faire correspondre l'adresse IP à un pays ou une ville sur une carte. Il a plusieurs tentatives pour localiser l’adresse IP, mais ces tentatives sont limitées.
